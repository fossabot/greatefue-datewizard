## Quick-start

### Installation

```bash
$ yarn add greatefue-datewizard
$ npm i greatefue-datewizard
```

### Usage

In your .ts or .js file, 


1. import the package

```ts
import { DaysAgo } from 'greatefue-datewizard';
```

2. Call the function(s)

```ts
var i = DaysAgo("2020-01-01");
```
